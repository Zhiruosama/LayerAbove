# LayerAbove

面向 **Adobe After Effects** 的小工具：在**当前选中图层上方**快速创建**等长**的调整类图层或固态层，或在**时间指示器（CTI）**位置插入**单帧**调整层。脚本为 **ExtendScript（.jsx）**，带 **ScriptUI** 面板界面。

**仓库**：<https://github.com/Zhiruosama/LayerAbove>

## 能做什么

| 能力 | 说明 |
|------|------|
| 等长调整图层 | 在参考图层**正上方**新建一层，**入点 / 出点**与参考层一致；全合成尺寸；效果为调整层。 |
| 等长白色固态层 | 同上，但为**白色全尺寸固态层**（非调整层）。 |
| 单帧调整图层 | 在**当前时间**插入**仅一帧**的调整层；可选中图层以决定叠放顺序。 |

所有操作均包在 **撤销组（Undo Group）** 中，便于一次撤销。

---

## 环境要求

- **Adobe After Effects**（需支持脚本中的 `ScriptUI`、`CompItem`、`AVLayer.adjustmentLayer` 等 API）。
- 脚本未使用 CEP / Node，**无需**额外扩展运行时。

> **说明**：部分旧版 AE 的脚本接口中**没有** `LayerCollection.addAdjLayer`。本仓库实现改为：`addSolid` 全画面固态 + `adjustmentLayer = true`，与「调整图层」在合成中的用途等效。

## 安装方式

### 方式 A：作为可停靠面板（推荐）

1. 将仓库中的 **`panels/LayerAboveTools.jsx`** 复制到 AE 的 **ScriptUI Panels** 目录。
2. **完全退出** After Effects 后重新启动。
3. 菜单栏：**窗口（Window）** 下找到与脚本同名的面板（如 **LayerAboveTools**），勾选打开。

**Windows** 典型路径（版本号与安装盘符请按本机修改）：

```text
C:\Program Files\Adobe\Adobe After Effects 20xx\Support Files\Scripts\ScriptUI Panels\
```

**macOS** 典型路径：

```text
/Applications/Adobe After Effects 20xx/Scripts/ScriptUI Panels/
```

### 方式 B：临时运行（浮动窗口）

1. 菜单：**文件 > 脚本 > 运行脚本文件…**（File > Scripts > Run Script File…）
2. 选择 **`LayerAboveTools.jsx`**。

此时以**浮动面板**打开，不依赖 `ScriptUI Panels` 目录；适合试用或不想重启 AE 时快速验证。

## 界面与三个功能

面板标题区为 **「图层上方工具」**，副标题为 **「所选图层或当前时间」**。主体分两块标签：**所选图层上方**、**当前时间**。

### 1. 等长调整图层

- **条件**：已打开一个**合成**；时间轴上**至少选中一个图层**。
- **结果**：新建调整层；**时长**取参考层 `outPoint - inPoint`（若异常则内部仍会用合成帧长兜底）；**堆叠顺序**在参考层**之上**（`moveBefore` 到参考层前）。

### 2. 等长白色固态层

- **条件**：同上，须选中图层。
- **结果**：新建白色、**全合成尺寸**固态层；时长与参考层对齐；同样移到参考层上方。

### 3. 单帧调整图层

- **条件**：须打开合成。
- **可选**：若选中图层，新层会插在该层**上方**；**未选中**也可执行，仅在当前时间插入单帧层。
- **结果**：`inPoint = 当前时间`，`outPoint = 当前时间 + 一帧`。

## 行为细则（建议通读）

### 参考图层：多选时选谁？

时间轴上**多个图层被选中**时，脚本取「**堆叠上最靠上**」的那一个已选图层作为参考：

- 即：**图层索引最小**（在合成中越靠上）的已选层。
- 面板底部有静态说明文案，与之一致。

### 调整图层在旧版 AE 上的实现

- 使用 **`comp.layers.addSolid`** 创建与合成同宽高的固态层，再设 **`layer.adjustmentLayer = true`**。
- 图层在「时间线」里可能显示为固态层图标，但**合成内效果**为调整层逻辑。

### 撤销

每次点击按钮会 `beginUndoGroup` / `endUndoGroup`，撤销名称与操作对应（如「上方创建调整图层」等）。

## 界面与交互说明

- **按钮高度**：随面板**垂直高度**在约 **42～88 px** 间变化；过小时有默认高度，保证可点。
- **按钮文字大小**：随按钮高度在约 **12～18 pt** 间变化（受 `ScriptUI.newFont` 与系统字体影响，极端环境下可能回退为系统默认）。
- **悬停提示**：各按钮带 `helpTip`，鼠标悬停可查看完整说明。
- **浮动窗口模式**：默认建议尺寸约 **300×320**，最小约 **260×240**（仅「运行脚本文件」打开的窗口；停靠面板由 AE 布局决定）。

> **技术说明**：根容器未使用自定义 `onDraw` 全屏铺底，以避免部分 Windows / AE 组合下子控件不绘制的问题；布局避免中间区域「零高度」导致整块空白。

## 常见问题

**Q：面板打开后一片空白？**  
A：请使用本仓库**当前版本**脚本；确保已执行 `layout.layout(true)` / `resize`（本脚本已包含）。若仍异常，可尝试「运行脚本文件」方式对比是否为面板宿主问题。

**Q：点调整图层相关按钮报错「addAdjLayer 未定义」？**  
A：请更新到本仓库版本；已改为 `addSolid` + `adjustmentLayer`。

**Q：提示先打开合成 / 先选中图层？**  
A：前两个按钮必须**有合成且有选中图层**；第三个按钮**仅需合成**，选中图层仅影响叠放顺序。

**Q：想改默认图层名 / 颜色？**  
A：直接编辑 `panels/LayerAboveTools.jsx` 中 `createAdjustmentLayer`、`addSolid` 的参数即可。
